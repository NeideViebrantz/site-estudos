# 📝 Repositório de Estudos - Curso SCTech

Este repositório contém exercícios e projetos desenvolvidos durante o curso **SCTech**.  
O objetivo é praticar **HTML e CSS**, explorando conceitos básicos de desenvolvimento web.

---

## 📌 Conteúdo do repositório

- Estruturação de páginas com **HTML**
- Estilização com **CSS** (incluindo display, flexbox, grid e animações)
- Criação de formulários e tabelas
- Estudos sobre semântica HTML e organização de conteúdo

---

## 🛠 Tecnologias utilizadas

- HTML5  
- CSS3
